# AIMA assignment — exercises

Each chapter has exercise files in `ch01/`, `ch02/`, … with the form convention:

- **Question** in `<!-- question -->` … `<!-- /question -->`
- **Your answer** in `<!-- answer -->` … `<!-- /answer -->`

Filenames may be `ex_N.md` or `ex_N.form.md`; the `.form.md` suffix marks a form-filled exercise so the AIMA Exercise Forms extension and tooling recognize it.

Fill in the answer blocks and push. You can edit the `.md` files directly or use the **AIMA Exercise Forms** extension in the sidebar (when opened in Codespaces/Cursor).

These files are generated from the course repo. To refresh them (e.g. after updating question text), from the **aima** repo run:

```bash
npm run exercise-forms:generate:template
```
